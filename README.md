==長生劍(Squid)==

技术特性

* Squid缓存技术
* 类型: HTTP代理
* 支持obfsproxy混淆

部署

```
wget -N --no-check-certificate  https://git.io/vXTHY  -O ./SLSrv.sh && bash SLSrv.sh 

```
支持平台
------------

* Windows
* IOS
* Android
* OSX

==孔雀翎(Anyconnect)

技术特性

* Ocserv
* 类型VPN
* 支持RSA非对称加密
* 支持椭圆曲线(ECC)非对称加密

部署

```
wget -N --no-check-certificate  https://git.io/vD69Z  -O ./ocserv.sh && bash ocserv.sh 

```
支持平台
------------

* Windows
* IOS
* Android
* OSX
